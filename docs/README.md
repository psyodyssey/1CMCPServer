# docs/

Документация проекта 1C Agent Platform.

Здесь собирается вся письменная часть платформы: архитектурные решения,
спецификации инструментов MCP, требования безопасности, эксплуатационные
инструкции и описание публичного API.

Позже здесь появятся разделы:

- `architecture/` — архитектурные решения и диаграммы;
- `tools-spec/` — спецификации MCP-инструментов (read / write / intelligence);
- `safety/` — правила безопасности, политики, ограничения;
- `runbooks/` — пошаговые инструкции эксплуатации и реагирования на инциденты;
- `api/` — описание внешнего API платформы.
