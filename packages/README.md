# packages/

Переиспользуемые библиотеки платформы, от которых зависят приложения из `apps/`.

Здесь живёт общая логика — всё, что не должно дублироваться между read / write /
intelligence серверами: транспорт MCP, запуск процессов 1С, политики безопасности,
аудит, здоровье системы, диагностика.

Позже здесь появятся:

- `mcp-common` — общая обвязка для MCP-серверов;
- `onec-process-runner` — запуск и управление процессами 1С;
- `onec-policy-engine` — политики допустимых операций;
- `onec-audit` — журналирование и аудит действий агента;
- `onec-health` — проверки состояния инфобазы и окружения;
- `onec-troubleshooting` — сбор и анализ диагностической информации.
