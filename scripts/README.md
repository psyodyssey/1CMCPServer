# scripts/

Вспомогательные скрипты для жизненного цикла платформы.

Сюда складываются скрипты разработчика, прогона тестов и сборки релизов —
всё, что помогает единообразно запускать повторяющиеся операции на локальной
машине и в CI.

Позже здесь появятся:

- `dev/` — скрипты для локальной разработки и запуска окружения;
- `test/` — скрипты прогона юнит- и интеграционных тестов;
- `release/` — скрипты сборки, версионирования и выпуска пакетов.
