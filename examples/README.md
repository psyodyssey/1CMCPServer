# examples/

Примеры и демонстрационные материалы для платформы 1C Agent Platform.

Здесь будут лежать учебные и тестовые артефакты: демо-инфобазы, готовые
дампы конфигураций и примеры патчей, которые можно скармливать агентам
для отладки, прогонов и наглядной демонстрации возможностей платформы.

Позже здесь появятся:

- `demo-infobase/` — демонстрационные инфобазы 1С;
- `demo-dumps/` — готовые дампы конфигураций для тестов;
- `sample-patches/` — примеры изменений, применяемых через write-сервер.
